==============================================================
HIGH-LEVEL GAME SUMMARY
==============================================================

Title: Barn Swarm Sniper
Genre: First-person static-position mobile sniper / precision shooter.
Theme: Professional barn rat pest control using realistic
       thermal and night-vision optics.

Core Loop:
  Scan → aim (gyro + touch) → stabilize → shoot → score →
  earn pellets → upgrade optics → unlock harder barns.

Key pillars:
  - Hybrid gyro + touch controls.
  - High-density rat swarms in small spaces.
  - Authentic NV/thermal scope look based on real footage.
  - Endless procedural barn layouts and infestations.

==============================================================
VISUAL & UX REFERENCE (REAL FOOTAGE MAPPING)
==============================================================

Visual sources to emulate:
  - vidral.id thermal/NV rat control reels
  - PigPatrol YouTube shorts (white-hot thermal, beam rats)
  - Green Guard Pest Control night-scope videos

Scope presentation:
  - Central circular (or oval) scope view with vignette.
  - HUD overlays: zoom "14.0x", REC icon, time, distance tag,
    kill count, scope mode.
  - Four major imaging modes, unlocked over time:
    1) Daylight
    2) Night Vision (green phosphor)
    3) Thermal White Hot
    4) Thermal Green

Rats:
  - Small, low-profile silhouettes, hugging walls, beams,
    under machinery, in crawlspaces.
  - Eye-shine dots in NV/thermal modes (bright white/green).

---------------------------
INPUT (HYBRID GYRO + TOUCH)
---------------------------

Scripts:
  - Input/GyroInputHandler.cs
  - Input/TouchInputHandler.cs
  - Input/InputSmoothingPipeline.cs
  - Input/HybridInputController.cs

Requirements:
  - Use UnityEngine.InputSystem if available; otherwise use
    Input.gyro and basic touch.
  - Gyro: map device rotation to yaw/pitch deltas.
  - Dead zone, EMA smoothing, sensitivity scalar per axis,
    max angular velocity clamp.
  - Touch: right-half drag to add deltas, tap-to-fire.

Aim assist (implemented in HybridInputController & WeaponController):
  - Reticle friction near rat colliders.
  - Soft magnetism when firing.
  - Slightly enlarged collider / hitbox.

---------------------------
CAMERA & SCOPE
---------------------------

Scripts:
  - Camera/SniperCameraController.cs
  - Camera/ZoomController.cs
  - Camera/BreathingSway.cs

Features:
  - Camera is parented to PlayerRig; rotation driven solely
    by HybridInputController.
  - Zoom levels per optics tier (2x–14x).
  - Smooth zoom interpolation (0.2–0.4s).
  - Optional sway (sine-based).
  - Recoil: vertical kick + recovery.

Scope overlay:
  - UI/ScopeOverlayController.cs
  - Draws vignette, reticle, zoom, mode, timer, kill count.
  - Supports multiple reticle styles and imaging modes, using
    URP render features / post-processing where needed.

RAT AI & OBJECT POOLING
---------------------------

Scripts:
  - AI/RatFSM.cs
  - AI/RatAIManager.cs
  - AI/MovementPathfinder.cs
  - AI/ScatterBehavior.cs
  - Pooling/RatObjectPool.cs
  - Pooling/EffectsObjectPool.cs

RatFSM states:
  - Idle, Move, Pause, Scatter, Dead, Pooled.
  - All transitions and guards as described earlier:

Object pool:
  - Pre-warm up to maxRatCount (e.g., 60).
  - Provide Get() / Return() methods.
  - No Instantiate/Destroy during gameplay.

---------------------------
LEVEL GENERATION & SPAWNING
---------------------------

Scripts:
  - Level/EnvironmentTileLibrary.cs
  - Level/LevelGenerator.cs
  - Level/SpawnZoneManager.cs
  - Level/LightingModeController.cs
  - Data/LevelSeedGenerator.cs

Environment tiles:
  - Prefab-based modular pieces:
    DaylightBarn, AfternoonBarn, LowLightBarn, Hayloft,
    Crawlspace, StorageCorridor, IndustrialBasement.

LevelGenerator:
  - Deterministic: uses LevelSeedGenerator (player level +
    date + device ID) to choose:
    - Tile variant
    - Lighting mode
    - Active spawn zones (2–5)
    - RatConfig (density, speed, pause times)
    - Objective (time limit / kill count / wave)
  - Instantiates the chosen tile prefabs and configures
    SpawnZoneManager with zone transforms.

SpawnZoneManager:
  - Holds a list of spawn zones (Transform + config).
  - For each wave, chooses zones and spawns rats from pool
    respecting max active limit.
  - Schedules spawn timings, wave completion.

---------------------------
WEAPON, HIT DETECTION, SCORING
---------------------------

Scripts:
  - Weapon/WeaponController.cs
  - Weapon/OpticsTierConfig.cs
  - Weapon/RecoilSystem.cs
  - Scoring/ScoreManager.cs
  - Scoring/ComboTracker.cs
  - Scoring/CurrencyManager.cs
  - UI/HitMarkerFeedback.cs
  - UI/HUDController.cs

WeaponController:
  - Uses raycasts from camera center (Physics.Raycast).
  - Checks rat colliders layer.
  - Applies aim-assist nudge before raycast.
  - Handles fire rate, recoil, audio cues.

ScoreManager:
  - Listens for OnRatKilled events.
  - Applies base score + multipliers for headshot, distance,
    speed kill, combo streak.
  - Emits events to HUDController for display.

ComboTracker:
  - Maintains streak; reset on miss or timeout.

CurrencyManager:
  - Converts score to pellets.
  - Handles spending on optics, scope modes, barn unlocks.
